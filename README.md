# Cumpleaños de Patty 💙

Sitio estático para sorprender a Patty en su cumpleaños.

## Cómo personalizar

### 1. Fotos
- Pon las imágenes dentro de `images/`
- Edita `script.js` y agrega los nombres en la lista `PHOTOS`:
  ```js
  const PHOTOS = [
    "images/foto1.jpg",
    "images/foto2.jpg",
  ];
  ```

### 2. Música (Romeo Santos)
- Guarda el mp3 dentro de `music/` con el nombre `romeo.mp3`
- O cambia el nombre en `index.html` (`<source src="...">`)

## Cómo verlo en local
Solo abre `index.html` en el navegador (doble click).

## Cómo subirlo gratis a GitHub Pages
1. Crea un repo nuevo en GitHub (ej: `patty-birthday`).
2. Sube todos estos archivos al repo.
3. Ve a **Settings → Pages**.
4. En "Source" elige la rama `main` y carpeta `/ (root)`. Guarda.
5. Espera 1 minuto y abre la URL que te muestra (algo como
   `https://tu-usuario.github.io/patty-birthday/`).

¡Listo! 💙
